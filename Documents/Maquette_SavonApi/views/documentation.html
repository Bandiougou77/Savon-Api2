Documentation

Cette application permet de simuler les caractéristiques d’une recette de savon afin de réduire le gaspillage de matières premières.

Fonctionnement

L’utilisateur saisit les pourcentages d’huiles végétales. Le système calcule les scores estimés.

Public cible

Savonniers, artisans et laboratoires travaillant avec ADEPRO.
